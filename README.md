# Fruit Picker

Моя первая игра!  
Классика жанра "арканоид", где нужно ловить фрукты и избегать бомб и гнилых плодов. 
Иногда вылетает муха и мешает это делать
После проигрыша муха аплодирует игроку — потому что каждый проигрыш — тоже опыт.

## Основные механики:
- Управление корзиной (влево/вправо)
- Спавн фруктов и враждебных объектов
- Подсчёт очков и рекордов
- Анимация мухи при проигрыше

Сделано с душой.  
И с мухой
